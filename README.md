# 🌧️ Kerala Flood Risk Predictor

An end-to-end ML project that predicts flood risk level 
(Low / Medium / High) for Kerala based on seasonal rainfall patterns.
# Test
Test of the model live here : https://karthikunni05-kerala-flood-predictor.hf.space/

## What I built
- Exploratory data analysis on 115 years of Kerala rainfall data
- Feature engineering (seasonal totals, monsoon ratio, lag features)
- Gradient Boosting classifier with F1 score of 0.847
- Streamlit web app with confidence visualization

## Tech Stack
Python · Pandas · Scikit-learn · Streamlit · Matplotlib

## Phases
- Phase 1: EDA & label creation
- Phase 2: Feature engineering  
- Phase 3: Model training & leakage detection
- Phase 4: Hyperparameter tuning
- Phase 5: Deployment
