# 🌊 Kerala Flood Risk Prediction System

> 🚀 A machine learning-powered web application that predicts flood risk levels in Kerala using over **100+ years of rainfall data**.

---

## 📌 Overview

Floods in Kerala have caused massive damage in recent years. This project uses **historical meteorological data** and **machine learning** to predict the likelihood of flood risk based on seasonal rainfall patterns.

💡 Built as a **real-world AI application** combining:

* Data Science
* Machine Learning
* Web Deployment

---

## 🎯 Key Features

✨ Predict flood risk in real-time
📊 Interactive input for rainfall parameters
📈 Visual analytics & probability insights
⚡ Fast and responsive Streamlit UI
🌐 Deployed on Hugging Face Spaces

---

## 🧠 Machine Learning Model

* 📌 Algorithm: **Gradient Boosting Classifier**
* 📊 Dataset: Kerala rainfall data (1901–2015)
* 🔍 Features used:

  * Monsoon rainfall
  * Pre/Post monsoon rainfall
  * Winter rainfall
  * Monsoon variability
  * Peak rainfall
  * Previous year rainfall

📈 **Performance:**

* F1 Score: **0.84+**
* Cross-validation: **5-Fold Stratified**

## 📊 How It Works

1. User inputs rainfall data
2. Features are processed
3. ML model predicts flood risk level:

   * ✅ Low
   * ⚡ Medium
   * ⚠️ High
4. Results + probability visualizations are displayed
